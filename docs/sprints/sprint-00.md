# Sprint 1 — Backend Foundation

# Sprint goal

Establish a production-ready Spring Boot backend foundation following Clean Architecture principles.

# OPS-1 - Initialize Spring Boot Project

# User Story

As a backend developer, I want to initialize the OpsPilot backend project so that future development is built on a stable and maintainable foundation.

---

# Description

Create the initial Spring Boot application using Gradle and Java 21.

The project should be able to build and start successfully without implementing any business functionality. This issue focuses only on establishing the project foundation.

---

# Acceptance Criteria

* Spring Boot project is created.
* Java 21 is configured.
* Gradle build completes successfully.
* The application starts without errors.
* The project can be opened and run from IntelliJ IDEA.
* The initial project structure is committed to the repository.

---

# Technical Notes

* Spring Boot 3.x
* Gradle (Kotlin DSL)
* No business logic or REST endpoints are required in this issue.

---

**2 Story Points**
