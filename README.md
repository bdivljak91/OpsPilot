# OpsPilot

Production-ready Spring Boot backend demonstrating applied AI through intelligent agents, tool calling, and workflow automation.

## Overview

OpsPilot is an AI-powered operations platform for payment monitoring, incident management, and intelligent workflow automation.

The project starts as a clean backend system and gradually evolves into an applied AI platform where agents can inspect data and execute approved backend tools.

## Planned Features

- Customer management
- Subscription management
- Payment tracking
- Incident management
- Ticket management
- Audit logging
- JWT authentication
- OpenAPI documentation
- PostgreSQL + Flyway migrations
- Dockerized local environment
- AI agent with tool calling

## Tech Stack

- Java 21
- Spring Boot 4.1.0 
- PostgreSQL
- Flyway
- Docker
- OpenAPI / Swagger
- GitHub Actions
- OpenAI API
